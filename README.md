# 📊 Data Science Salary Dashboard

A full-stack analytical tool designed to visualize global salary trends in the data sector. Built with Python, this interactive dashboard provides real-time insights into compensation across different seniorities, roles, and locations.

### Key Features
* **Interactive KPIs:** Instant view of average and maximum salaries.
* **Global Mapping:** Choropleth maps showing Data Scientist pay by country.
* **Dynamic Filters:** Sidebar navigation to drill down by company size, experience level, and year.
* **Data Visuals:** High-quality charts powered by Plotly Express.

### Tech Stack
* **Language:** Python
* **Web Framework:** Streamlit
* **Data Processing:** Pandas & NumPy
* **Visualization:** Plotly

### How to Run
1. Install dependencies: `pip install streamlit pandas plotly`
2. Launch the app: `streamlit run app.py`
